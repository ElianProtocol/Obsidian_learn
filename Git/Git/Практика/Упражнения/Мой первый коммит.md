
## Git Commit 01

### Цель

Узнайте, как фиксировать изменения в репозиториях Git.
### Инструкции

1. Создайте новую директорию.
2. Создайте репозиторий Git.
3. Создайте новый файл `file`с именем и содержимым "hello commit".
4. Зафиксируйте новый файл.
5. Выполните команду `git`, чтобы убедиться, что ваш коммит был записан.

### После выполнения упражнения

Ответьте на следующие вопросы:

- В чём преимущества коммитов?
- Есть ли другой способ проверить, был ли создан коммит?
  

Решение:

## Git Commit 01 - Solution

```
mkdir my_repo && cd my_repo
git init
echo "hello_commit" > file
git add file
git commit -a -m "It's my first commit. Exciting!"
git log
```
